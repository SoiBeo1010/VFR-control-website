🚀 QUICK START INSTRUCTIONS

1. Install Dependencies:
   pip install -r requirements.txt

2. Create Database:
   mysql -u root -p
   CREATE DATABASE qms_system;
   EXIT;

3. Configure Environment:
   cp .env.example .env
   [Edit .env with your credentials]

4. Start Server:
   python main.py

5. Access System:
   http://localhost:8000
